Stack Attack

The goal of this project was to creating a captivating arcade experience with a thrilling rendition of the game Stacker. Our aim was to transport users into the nostalgic ambiance of an arcade, providing them with an exceptional and enjoyable time.

Using the PIC32 microcontroller used in class along with the capacitive breakout touch board, PING sensor, and a custom-built LED array and Game Rig, we built our physical arcade machine.
